# Brainmade

## About

Разработка стильных сайтов и приложений для вашего бизнеса. \
Упакуем ваш бизнес в креативный сайт, который будет приносить клиентов.

## Tech

- React.js
- Next.js
- TypeScript
- ESLint
- Stylelint
